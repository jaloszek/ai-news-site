### Claude Code

- 🛠 [Auto mode for Claude Code](https://claude.com/blog/auto-mode) — A new permissions mode where a background Sonnet 4.6 classifier reviews each tool call before execution, blocking actions that exceed scope (mass deletions, data exfiltration, malicious code). Middle ground between manual approvals and `--dangerously-skip-permissions`.
  - Research preview for Team plan; Enterprise and API coming soon. Toggle with Shift+Tab in CLI
