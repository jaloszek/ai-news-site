# AI News: Week of March 30, 2026

## Monday, March 30

### Claude Code

- 🛠 [CC 2.1.88: Flicker-free rendering, PermissionDenied hook, and critical stability fixes](https://code.claude.com/docs/en/changelog) — New `CLAUDE_CODE_NO_FLICKER=1` env var enables alt-screen rendering with virtualized scrollback, eliminating UI jitter in iTerm2 and tmux. The `PermissionDenied` hook fires after auto mode classifier denials, letting models retry with `{retry: true}` for smoother agentic workflows.
  - Named subagents now appear in `@` mention typeahead; hooks' `if` condition filtering fixed for compound and env-var-prefixed commands
  - Critical fixes: prompt cache misses in extended sessions, nested CLAUDE.md re-injection during long sessions, Edit/Write doubling CRLF on Windows, StructuredOutput ~50% failure rate in multi-schema workflows, memory leak from large JSON in LRU cache, OOM crash on >1 GiB files
  - Voice mode fixed on macOS Apple Silicon and Windows terminals; long `/btw` responses now render in scrollable viewports
