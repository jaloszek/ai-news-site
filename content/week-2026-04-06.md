# AI News: Week of April 6, 2026

## Monday, April 6

### Anthropic

- 🔥 [Anthropic expands partnership with Google and Broadcom for multiple gigawatts of next-generation compute](https://www.anthropic.com/news/google-broadcom-partnership-compute) — Anthropic announced a major expansion of its compute partnership with Google and Broadcom, securing access to multiple gigawatts of next-generation infrastructure. This signals a significant scale-up in training and inference capacity for future Claude models.
  - Larger compute commitments typically precede major model capability jumps — relevant for developers planning long-term integrations.

## Tuesday, April 7

### Claude Code

- 🛠 [Claude Code v2.1.94](https://github.com/anthropics/claude-code/releases/tag/v2.1.94) — Major release adding Amazon Bedrock Mantle support (`CLAUDE_CODE_USE_MANTLE=1`), a significant update for teams running Claude via AWS infrastructure.
  - Practical bump: default effort level bumped from medium to **high** for API-key, Bedrock/Vertex/Foundry, Team, and Enterprise users — expect more thorough responses by default without changing any config.

- 🛠 [Slack MCP compact header + plugin `keep-coding-instructions`](https://github.com/anthropics/claude-code/releases/tag/v2.1.94) — Slack MCP send-message calls now display a compact `Slacked #channel` header with a clickable channel link; plugin authors gain a `keep-coding-instructions` frontmatter field to control output styles.
  - Also fixes agents stuck after 429 rate-limit responses, Console login on macOS keychain, hyperlink double-opening, CJK text corruption, and multi-worktree `--resume`.

### Anthropic

- 🔥 [Project Glasswing — Anthropic leads industry coalition to secure critical software](https://www.anthropic.com/news/project-glasswing) — Anthropic published the Claude Mythos Preview System Card (244 pages) and launched Project Glasswing, a security initiative uniting AWS, Apple, Microsoft, Google, NVIDIA, Cisco, CrowdStrike, and others to fix critical software vulnerabilities. Anthropic is withholding Mythos Preview from public release after the model demonstrated the ability to discover and chain thousands of zero-day vulnerabilities across major operating systems and browsers.
  - Developers building security tooling should study the Mythos Preview system card for new capability benchmarks — this sets a new bar for AI-assisted vulnerability research.

## Wednesday, April 8

### Claude Code

- 🛠 [Claude Code v2.1.96 — Bedrock auth hotfix](https://github.com/anthropics/claude-code/releases/tag/v2.1.96) — Fixes a regression introduced in v2.1.94 where Bedrock requests failed with `403 "Authorization header is missing"` when using `AWS_BEARER_TOKEN_BEDROCK` or `CLAUDE_CODE_SKIP_BEDROCK_AUTH`.
  - If you upgraded to v2.1.94 and saw Bedrock 403 errors, update to v2.1.96 immediately.

### Anthropic

- 🔥 [Anthropic restricts third-party harnesses (OpenClaw, Paperclip) from subscription limits](https://www.youtube.com/watch?v=LXiEK5xumSg) — Effective April 4, Anthropic began cutting off third-party Claude wrappers that bypass subscription rate limits, including OpenClaw and Paperclip. The move drew backlash from users who relied on these tools for higher throughput.
  - Developers depending on unofficial harnesses for capacity management should plan migration to the official API or supported SDK integrations.
