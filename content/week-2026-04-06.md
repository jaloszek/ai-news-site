### Claude Code

- 🔥 [Claude Mythos Preview: Anthropic holds back new frontier model, launches Project Glasswing](https://www.anthropic.com/glasswing) — Anthropic's new Claude Mythos Preview model demonstrates extraordinary cybersecurity capabilities — finding zero-day vulnerabilities in every major OS and browser, including bugs that survived 16–27 years of human review — but will not be made generally available. Instead, Anthropic is deploying it defensively to 12 trusted partners (AWS, Apple, Google, Microsoft, Nvidia, Cisco, CrowdStrike, JPMorganChase, Linux Foundation, Palo Alto Networks, and others) through Project Glasswing, backed by a $100M compute credit pledge and $4M for open-source security organizations.

Canadian banks and the U.S. Treasury held emergency meetings over implications.

### AI World

- 🔵 [Google Gemma 4: Byte for byte, the most capable open models](https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/) — Google DeepMind released Gemma 4 on April 2 under Apache 2.0 (unrestricted commercial use). Four sizes: 2B, 4B, 26B MoE, and 31B Dense. All are natively multimodal (text + images; audio on edge models), 256K context, 140+ languages. The 31B Dense ranks #3 on the Arena AI open-model leaderboard; 26B MoE fits in 14GB and runs at 85 tokens/s on consumer hardware. Fully offline-capable.
  - Practical for Claude Code users: Gemma 4 via Ollama can serve as a fast, free local backend; the Apache 2.0 license removes friction for production deployment
  - Also: [Google Cloud availability](https://cloud.google.com/blog/products/ai-machine-learning/gemma-4-available-on-google-cloud), [DeepMind model page](https://deepmind.google/models/gemma/gemma-4/)

- 🔵 [Google I/O 2026 announced for May 19–20](https://nationaltoday.com/us/ca/mountain-view/news/2026/04/11/google-i-o-2026-announced-major-ai-and-android-reveals/) — Google confirmed I/O 2026 at Shoreline Amphitheatre, Mountain View; Gemini AI updates and Android 17 are the expected headliners. Gemini 3 Flash is now the default model in the Gemini app, replacing Gemini 2.5 Flash.
  - Typically the venue for Google's biggest model releases and developer platform changes — worth watching for Gemini 4 and Vertex AI announcements

- 🟡 [MiniMax M2.7 open-sourced with free API key](https://www.minimax.io/news/minimax-m27-en) — MiniMax released M2.7, a "self-evolving" agent model: within MiniMax's own RL research workflows, M2.7 handles 30–50% of the pipeline end-to-end. Benchmarks: 56.22% on SWE-Pro (matches GPT-5.3-Codex), 57.0% on Terminal Bench 2, #1 ELO among open-source models on GDPval-AA. Weights on HuggingFace; runs on Ollama; free API available.
  - Strong coding/agentic benchmark scores and a free API make this worth testing as a Claude Code alternative for budget-sensitive pipelines
  - Also: [MarkTechPost writeup](https://www.marktechpost.com/2026/04/12/minimax-just-open-sourced-minimax-m2-7-a-self-evolving-agent-model-that-scores-56-22-on-swe-pro-and-57-0-on-terminal-bench-2/), [OpenRouter](https://openrouter.ai/minimax/minimax-m2.7)
