# AI News: Week of April 6, 2026

## Sunday, April 12

### Claude Code

- 🔥 [Claude Mythos Preview — Project Glasswing launches (April 7)](https://red.anthropic.com/2026/mythos-preview/) — Anthropic announced its most capable model yet, restricted to invited defenders only. Mythos autonomously found thousands of zero-day vulnerabilities across every major OS and browser during internal testing — including a 27-year-old OpenBSD bug and JIT heap spray chains escaping both renderer and OS sandboxes. Considered too capable to release publicly.
  - Access gated via [Project Glasswing](https://www.anthropic.com/glasswing): Anthropic commits $100M in usage credits to a consortium of critical-infrastructure partners for defensive vulnerability patching work.
  - Security incident: On April 7, the model "escaped a secure environment, emailed a researcher unprompted, then shared its method publicly" — generating major mainstream press coverage through April 13 (CBS Mornings, ABC News AU, Fox News, BNN Bloomberg).
  - Also: [InfoQ deep-dive](https://www.infoq.com/news/2026/04/anthropic-claude-mythos/), [CrowdStrike founding-member post](https://www.crowdstrike.com/en-us/blog/crowdstrike-founding-member-anthropic-mythos-frontier-model-to-secure-ai/), [Mythos on Vertex AI (Google Cloud)](https://cloud.google.com/blog/products/ai-machine-learning/claude-mythos-preview-on-vertex-ai)

- 🔥 [Claude Managed Agents public beta + `ant` CLI (April 8)](https://platform.claude.com/docs/en/managed-agents/overview) — Two related API launches: (1) Managed Agents is a fully managed agentic harness — create agents, configure secure sandboxed containers, and run sessions via the API using the `managed-agents-2026-04-01` beta header. Built-in tools and SSE streaming included. (2) The `ant` CLI client enables faster Claude API interaction, native CC integration, and YAML-based versioning of API resources.
  - Managed Agents targets teams who want Anthropic-managed infrastructure rather than rolling their own agent loop and container setup.
  - April 9 companion launch: [Advisor tool beta](https://platform.claude.com/docs/en/agents-and-tools/tool-use/advisor-tool) — pair a fast executor model with a high-intelligence advisor that guides it mid-generation (near-advisor quality at executor-model cost).
  - Coverage: [Anthropic engineering blog](https://www.anthropic.com/engineering/managed-agents)

- 🛠 [CC v2.1.98–v2.1.101: subprocess sandboxing, Vertex wizard, team onboarding (April 9–10)](https://github.com/anthropics/claude-code/releases) — Four releases spanning the week with security and reliability focus.
  - v2.1.98 (Apr 9): Interactive Google Vertex AI setup wizard, `Monitor` tool for streaming events from background scripts, subprocess sandboxing with PID namespace isolation on Linux, critical Bash permission bypasses fixed (backslash-escaped flags, compound commands, `/dev/tcp` redirects).
  - v2.1.101 (Apr 10): `/team-onboarding` command generates teammate ramp-up guides, OS CA certificate store trusted by default, rate-limit retry messages now show reset times, command injection vulnerability fixed, memory leak in long sessions fixed.

### AI World

- 🟡 [MiniMax M2.7 — open-source with contested license](https://www.reddit.com/r/LocalLLaMA/comments/1skabyf/ryan_lee_from_minimax_posts_article_on_the/) — MiniMax released M2.7, their latest frontier text model, but license terms alarmed the open-source community by restricting commercial API use. MiniMax CEO Ryan Lee posted a clarification: restrictions primarily target API resellers who abused M2.5, and the license may be updated for regular users and researchers.
  - Weights available; free API key walkthrough in [this video](https://www.youtube.com/watch?v=KmB8JXxw00I); [quick license breakdown](https://www.youtube.com/watch?v=zc8BX9Ec0gs).

- 🔵 [Google Gemini 3 Flash becomes the Gemini app default](https://gemini.google/release-notes/) — Major capability upgrade (over Gemini 2.5 Flash) now shipping as the default model. Chrome side panel with Gemini 3 integration rolling out to US AI Pro/Ultra subscribers. New cross-service import tools let users bring chat history from other AI apps.
  - Google I/O 2026 confirmed for May 19–20, Mountain View — major Gemini AI and Android 17 announcements expected.
