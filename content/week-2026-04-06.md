## Friday, April 10

### Claude Code

- 🛠 **Apr 10** — [Claude Code v2.1.101: Team onboarding command, enterprise TLS zero-config, security fixes](https://github.com/anthropics/claude-code/releases) — A broad maintenance-and-polish release. The headline feature is `/team-onboarding`, which generates a personalized ramp-up guide for new teammates by analyzing your local Claude Code usage patterns. Enterprise TLS is now zero-config: the OS certificate store is trusted by default, so corporate HTTPS proxies just work without extra configuration.
  - Also: `/ultraplan` and remote-session features now auto-provision cloud environments; rate-limit errors now show exact reset times; `claude -p --resume` accepts session titles; `/rename` sessions work correctly; unrecognized hook event names no longer silently discard entire settings files
  - Security: fixed a **command injection vulnerability in the POSIX `which` fallback**, multiple permission rule enforcement gaps, and memory leaks. Also: [Claude Code changelog](https://code.claude.com/docs/en/changelog), [Releasebot](https://releasebot.io/updates/anthropic/claude-code)

### AI World

- 🔵 **Apr 10** — [Google Notebooks: NotebookLM fully integrated into the Gemini app](https://blog.google/innovation-and-ai/products/gemini-app/notebooks-gemini-notebooklm/) — Google merged NotebookLM's research-assistant capabilities directly into Gemini. Users can create notebooks from the Gemini sidebar, uploading PDFs, docs, URLs, and YouTube videos into a searchable AI-powered repository that auto-syncs between both apps.
  - Available now for Google AI Ultra, Pro, and Plus subscribers on web; mobile, free tier, and some EU regions rolling out in coming weeks
  - Useful for developers doing deep research dives: cite sources, generate summaries, and get audio overviews — all within the same Gemini interface. Also: [Dataconomy](https://dataconomy.com/2026/04/10/google-integrates-notebooklm-into-gemini-app-for-all-users/), [gHacks](https://www.ghacks.net/2026/04/10/google-gemini-adds-notebooks-feature-with-automatic-notebooklm-sync-for-projects/)

- 🔵 **Apr 10** — [Veo 3.1 and Veo 3.1 Fast video models launch in public preview](https://ai.google.dev/gemini-api/docs/changelog) — Google's next video generation models are now in public preview on the Gemini API. New: extend existing Veo-created videos, reference up to 3 images to seed a clip, provide first/last frames for bookend-to-bookend generation. Output durations now include 4, 6, and 8-second options.
  - Also in the same update cycle: **Computer Use tool** added to gemini-3-pro-preview and gemini-3-flash-preview — Google's equivalent of Claude's computer use, now on their frontier preview models
  - Also: [Google Cloud release notes](https://docs.cloud.google.com/gemini/enterprise/docs/release-notes)
