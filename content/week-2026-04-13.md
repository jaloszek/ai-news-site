# AI News: Week of April 13, 2026

## Monday, April 13 (today)

### Claude Code

- 🔥 [Project Glasswing: Securing critical software for the AI era](https://www.anthropic.com/glasswing) — Anthropic officially unveils Claude Mythos Preview and Project Glasswing. Mythos is Anthropic's most capable model to date (93.9% SWE-bench Verified vs Opus 4.6's 80.8%; 83.1% on cybersecurity benchmarks vs 66.6%) — but it will NOT be publicly available. Instead, Anthropic launched a 12-member consortium (AWS, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorgan Chase, Linux Foundation, Microsoft, NVIDIA, Palo Alto Networks + Anthropic) with $100M in usage credits + $4M in open-source security donations to use Mythos purely for finding and patching vulnerabilities in critical infrastructure.
  - In testing, Mythos autonomously identified thousands of zero-days across every major OS and browser — including a 27-year-old OpenBSD flaw and a 16-year-old FFmpeg vulnerability that all prior automated tools had missed. It succeeded at generating working JavaScript shell exploits 181 times where Opus 4.6 succeeded only twice across hundreds of attempts.
  - Also: [Mythos Preview details (red.anthropic.com)](https://red.anthropic.com/2026/mythos-preview/), [InfoQ deep-dive](https://www.infoq.com/news/2026/04/anthropic-claude-mythos/), [AWS Bedrock gated research preview](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-bedrock-claude-mythos/), [Vertex AI](https://cloud.google.com/blog/products/ai-machine-learning/claude-mythos-preview-on-vertex-ai), [CrowdStrike partnership](https://www.crowdstrike.com/en-us/blog/crowdstrike-founding-member-anthropic-mythos-frontier-model-to-secure-ai/)

- 🛠 [Claude Code v2.1.101: Team onboarding, OS CA cert trust, focus mode improvements](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md) — Latest CC release (April 11). Adds `/team-onboarding` command that generates a teammate ramp-up guide from your local CC usage history. OS CA certificate store is now trusted by default, so enterprise TLS proxies work without extra configuration. Brief mode now retries once when Claude responds with plain text instead of structured output. Focus mode writes more self-contained summaries.
  - No new CC versions released in the last 24 hours; watch [releasebot](https://releasebot.io/updates/anthropic/claude-code) for v2.1.102+

### AI World

- 🟡 [MiniMax M2.7 open-weight model generating major community buzz](https://fazm.ai/blog/new-ai-model-releases-open-source-projects-april-2026) — MiniMax M2.7 is dominating developer YouTube today as a high-performing open-source coding model for agents. Available with a free API key. Community note: it is not fully open-source — weights are available but training code/data are withheld. Being tested as a backend for "OpenClaw" and self-hosted CC-adjacent toolchains. Scores competitively on coding benchmarks against Qwen 3 and Llama 4.
  - Multiple community builds show it running locally for multi-agent orchestration; the "not fully open" licensing is drawing pushback on r/LocalLLaMA
